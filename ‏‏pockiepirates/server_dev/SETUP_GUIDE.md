# מדריך הפעלה — Pockie Pirates Server

מדריך עבורך, כמשתמש לא־מפתח. הצעדים הם בסדר. **אל תקפוץ קדימה** — כל צעד מאמת שמשהו עובד לפני שעוברים לבא.

המטרה של המדריך הזה היא רק **שלב אחד**: שהשרת ירוץ, נראה שהוא חי, ויהיו לוגים שאפשר לחזור איתם לשיחה הבאה. החיבור של הקליינט עצמו לשרת מורכב יותר ויטופל בנפרד.

---

## שלב 1 — להתקין Python

Python היא שפת התכנות שבה השרת כתוב. צריך אותה רק כדי להריץ אותו, לא כדי לכתוב משהו.

### Windows

1. הולך ל־<https://www.python.org/downloads/>
2. לוחץ על הכפתור הצהוב הגדול "Download Python 3.X.X"
3. **חשוב:** במסך ההתקנה הראשון תסמן "Add python.exe to PATH" (תיבה למטה)
4. ממשיך עם Install Now

### Mac

1. פותח Terminal (Cmd+Space, מקליד "Terminal", Enter)
2. מקליד: `python3 --version` ו־Enter
3. אם זה מציג גרסה — סיימת. אם לא — מציע להתקין, מאשר ו־Mac יוריד אוטומטית

### לבדיקה (בכל מערכת):

פותח חלון פקודה חדש (Command Prompt ב־Windows, Terminal ב־Mac), מקליד:

```
python --version
```

ב־Windows, או:

```
python3 --version
```

ב־Mac/Linux. אמור להראות `Python 3.X.X`. אם זה עובד — סיימת את שלב 1.

## שלב 2 — להפעיל את השרת

1. פותח חלון פקודה חדש (Command Prompt ב־Windows, Terminal ב־Mac).

2. מתמקם בתיקייה הנכונה. **תעתיק־הדבק** את השורה הזאת בדיוק:

   **Windows:**
   ```
   cd /d "C:\path\to\pockie pirates\server_dev\skeleton"
   ```
   (תחליף את `C:\path\to\` בנתיב האמיתי לתיקייה שלך. הכי קל: גורר את התיקייה לתוך חלון הפקודה — Windows יעתיק את הנתיב אוטומטית.)

   **Mac:**
   ```
   cd "/Users/<username>/path/to/pockie pirates/server_dev/skeleton"
   ```
   (גם פה: גורר את התיקייה לתוך Terminal — הוא ימלא את הנתיב.)

3. מקליד:

   **Windows:**
   ```
   python server.py
   ```

   **Mac:**
   ```
   python3 server.py
   ```

4. אמור להופיע פלט שדומה לזה:

   ```
   [init] Loaded 58 game data tables, 19,122 records
   ======================================================================
     Pockie Pirates emulator
     Listening on http://0.0.0.0:8080/
     Logs: ...
   ======================================================================
   ```

   **חשוב:** השרת ימשיך לרוץ ויחסום את חלון הפקודה. **אל תסגור** את החלון — זה השרת. כל עוד החלון פתוח, השרת חי.

   אם כתוב "Address already in use" — משהו אחר משתמש בפורט 8080. סוגר את החלון, פותח חדש, ובמקום `python server.py` מקליד `python server.py 8888` (או כל מספר אחר). תזכור את המספר לשלב 3.

## שלב 3 — לבדוק שהשרת חי

זאת הבדיקה החשובה. **חלון הפקודה משלב 2 חייב להישאר פתוח.**

פותח את הדפדפן (Chrome/Firefox/Safari), מקליד בסרגל הכתובות:

```
http://127.0.0.1:8080/status
```

(אם השתמשת בפורט אחר בשלב 2, תחליף 8080 למספר שלך.)

אמור להופיע בדפדפן טקסט כמו:

```
Pockie Pirates emulator running.
Tables loaded: 58
Handlers registered: 17
Players in memory: 0
Logs: ...
```

**אם רואים את זה — השרת עובד.** זה הניצחון הראשון. שולח לי בשיחה הבאה צילום מסך של זה, ואני יודע שהבסיס מוכן.

אם רואים "This site can't be reached" או דף שגיאה — חוזר לשלב 2 ובודק שחלון הפקודה עדיין פתוח ושאין בו שגיאה אדומה.

## שלב 4 — איפה הלוגים

תוך כדי שהשרת רץ, הוא כותב כל מה שקורה לתיקייה הזאת:

```
pockie pirates/server_dev/logs/
```

בכל פעם שתפעיל את השרת, נוצרים שני קבצים חדשים שם, עם תאריך בשם:

- `transcript_YYYYMMDD_HHMMSS.jsonl` — תיעוד מלא של כל בקשה
- `unhandled_YYYYMMDD_HHMMSS.txt` — בקשות שהשרת לא ידע איך לטפל בהן

לבינתיים הלוגים יהיו ריקים כי שום קליינט לא מחובר לשרת. זה תקין. הלוגים יתמלאו ברגע שנחבר את הקליינט בשלב הבא.

## שלב 5 — לעצור את השרת

כשרוצה לסיים: בחלון הפקודה של השרת, **Ctrl+C**. השרת יתפנה.

## מה לעשות בשיחה הבאה

מצרף לי בשיחה הבאה:

1. **צילום מסך** של דף `/status` בדפדפן — מאשר שהשרת רץ אצלך
2. **צילום מסך** של חלון הפקודה — מראה שהוא עלה בלי שגיאות
3. אומר לי: "המשך מהשיחה הקודמת על Pockie Pirates, השרת רץ. השלב הבא — חיבור הקליינט."

ואני נמשיך משם — שלב חיבור הלקוח של פלאש זה נושא בפני עצמו ואצטרך לתת לך מדריך נפרד לזה (כי Flash Player כבר לא במצב פשוט בשנת 2026).

## שאלות נפוצות

**ש: כתבת בעבר שצריך לערוך קובץ hosts ולהתקין Flash Player. למה לא פה?**
ת: כי במדריך הקודם דחפתי הכל יחד וזה היה מבלבל. כאן רק שלב הראשון — שהשרת חי. שלבי החיבור של הקליינט יבואו אחר כך, אחרי שזה עובד.

**ש: לכמה זמן השרת ירוץ?**
ת: כל זמן שחלון הפקודה פתוח. אפשר לסגור ולפתוח מתי שרוצה — אין נזק, פשוט הוא נסגר.

**ש: צריך אינטרנט?**
ת: רק להתקנה הראשונית של Python. אחר כך השרת רץ לבד מקומית, בלי אינטרנט.

**ש: יש סיכון אבטחה?**
ת: השרת רץ רק בפורט 8080 בכתובת 127.0.0.1 (= המחשב שלך בלבד). הוא לא נגיש מבחוץ. אם רוצה זהירות מיוחדת, אפשר להריץ אותו רק כשצריך ולעצור אחרי.

**ש: למה רק 17 handlers ולא יותר?**
ת: בנינו את אלה לפי השמות שמצאנו בקליינט. ה־100 הנוספים יבנו אחרי שנראה במציאות מה הקליינט מבקש — לכן צריך את הלוגים שייווצרו כשנחבר את הקליינט.
